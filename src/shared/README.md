# Shared

Codigo compartilhado sem regra especifica de dominio.

Responsabilidades:

- componentes reutilizaveis;
- helpers pequenos;
- tipos utilitarios;
- constantes de interface;
- funcoes puras genericas.

Evitar transformar `shared` em deposito de qualquer coisa. Se um codigo pertence
a um dominio, ele deve ficar em `domain`.
