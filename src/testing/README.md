# Testing

Suporte de testes.

Estado atual em `v0.0.1`: os testes existentes cobrem o Playground de
referencia e o E2E do scaffold. Esta pasta deve apoiar os proximos testes de
dominio, persistencia e interface do MVP.

Responsabilidades planejadas:

- fixtures;
- builders;
- helpers de renderizacao;
- dados de exemplo para testes unitarios e de integracao.

Codigo desta pasta deve facilitar testes, nao virar dependencia de producao.
