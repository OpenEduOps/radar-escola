# Radar Escola

Aplicativo desktop local para escolas acompanharem necessidades operacionais.

Este repositorio sera o projeto de produto do Radar Escola dentro da
organizacao OpenEduOps.

## Estado Atual

Repositorio criado para receber o desenvolvimento do app.

O planejamento inicial, requisitos da V0 e scaffold tecnico temporario ainda
estao no repositorio `OpenEduOps/organizacao` e devem ser migrados de forma
controlada para ca.

## Direcao Do Produto

- Windows-first.
- Local-first.
- Tauri + React + TypeScript + SQLite.
- UX da pessoa usuaria final antes da tecnologia.
- Linguagem em Portugues Brasileiro.
- Sem internet obrigatoria na V0.

## Proximo Passo

Migrar para este repositorio a documentacao e o scaffold minimo do Radar Escola,
preservando historico e evitando duplicacao confusa entre repositorios.
